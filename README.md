<p align="center">FCPHP是一个轻量级、简单化的PHP框架。</p>

## 特点
- 多配置、配置分离
- 惰性加载、单例实例
- 强大的路由分发、支持正则
- 自己实现的DB操作、简单、快捷
- 多继承、便于扩展、支持事件监听
- 想不到了，看案例吧

### 安装

直接下载源码或者使用 composer 安装

````
{
    "require": {
        "lovefc/fcphp6": "0.0.1"
    }		
}
````
